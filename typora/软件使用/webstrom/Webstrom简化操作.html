    # 1快捷键生成自定义代码块

    操作步骤：

    # 1.1点击文件-->设置

    # 1.2搜索live-->找到活动模板-->打开HTML-->点击+-->添加模板

    # 1.3改快捷键-->加代码块-->所有文件格式都生效

    # 1.4设置完成后点击应用保存

    # 1.5使用时，在文档空白地方按 jq 再按 TAB 键

    # 2浏览器快捷键打开

    # 2.1打开设置setting

    # 2.2搜索keymap-->点击快捷键-->搜索default

    # 2.3找到用本地浏览器打开-->右击添加快捷键

    # 2.4按你所需要的快捷键
